# API Stubs

This file tracks backend API endpoints that the mobile app needs but are **not yet implemented** on the server.

Each entry should include:
- **Endpoint**: HTTP method + path
- **Description**: What it does
- **Request/Response shape**: Expected payloads
- **Added by**: Who/when it was stubbed

When a backend endpoint is implemented, remove the corresponding entry from this list.
